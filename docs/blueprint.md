# **App Name**: FixMyRide

## Core Features:

- User Authentication: Register and login using Firebase Authentication (Email/Phone/Google).
- Service Request Creation: Create a service request by selecting the problem type and adding vehicle details and descriptions.
- Live Location Sharing: Share live GPS location with the service provider.
- Real-time Request Broadcasting: Broadcast service requests in real-time using Firebase.
- Helper Matching and Navigation: Match nearby helpers using location data and enable navigation to the user's location.
- Service Status Updates: Update service status (Accepted → On the Way → Completed).
- Feedback and Rating: Submit feedback and rating after the service is completed.

## Style Guidelines:

- Primary color: Vibrant blue (#3498db) to represent reliability and trust.
- Background color: Light blue (#e7f5ff), a lighter shade of the primary to create a calming environment.
- Accent color: Orange (#e67e22) to highlight calls to action and important information, such as the 'request help' button.
- Body and headline font: 'PT Sans' (sans-serif) to combine a modern look and a little warmth or personality; well-suited for both headlines and body text
- Use clear and recognizable icons for service types and status indicators.
- Implement a clean and responsive layout optimized for both mobile and desktop.
- Use subtle animations to indicate real-time updates and status changes.
